Panduan Grup WhatsApp

Cara Mencari Kode Grup WhatsApp

  1. Buka WhatsApp dan pilih grup yang ingin dibagikan
  2. Tap menu (3 titik) → Info Grup
  3. Scroll ke bawah → Undang via Tautan Undangan
  4. Tap "Salin Tautan" → Tautan undangan akan disalin
  5. Tautan akan terlihat seperti: https://chat.whatsapp.com/xxxxx
  6. Kode grup adalah bagian terakhir setelah "/" (contoh: xxxxx)

Syarat & Ketentuan

  • ✓ Hanya admin yang bisa menambah kode grup
  • ✓ Setiap jabatan harus memiliki 1 grup WhatsApp
  • ✓ Karyawan otomatis bertemu invite link saat pertama kali masuk aplikasi
  • ✓ Laporan lembur, standby, share lokasi akan dikirim ke grup sesuai jabatan
  • ✓ Kode grup dapat diubah kapan saja
Panduan Link Database Rumah Sakit

Cara Membuat & Menggunakan Link Database RS

  1. Siapkan Google Sheet dengan data Rumah Sakit (Kode, Nama RS, Harga, Status)
  2. Share sheet dengan akses "Viewer" atau "Editor"
  3. Publish ke web: File → Share → Publish to web (pilih format CSV)
  4. Copy URL yang diberikan
  5. Paste URL di field "Link Database RS" di menu Rumah Sakit

Syarat & Ketentuan

  • ✓ Link harus mengarah ke file CSV/JSON yang valid
  • ✓ Format kolom: Kode | Nama RS | Harga | Status
  • ✓ Link dipublikasikan otomatis setelah disimpan
  • ✓ Data dari sheet akan di-sync otomatis setiap 1 jam
  • ✓ Admin bisa refresh manual kapan saja
