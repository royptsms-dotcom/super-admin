## 🆘 SUPPORT / TROUBLESHOOTING

### Issue: Font tidak berubah?
**Solusi**:
1. Clear browser cache
2. Check localStorage (`admin_prefs`)
3. Reload halaman
4. Check Google Fonts loading

### Issue: Warna tidak match?
**Solusi**:
1. Check CSS variables di line 11
2. Verify tidak ada inline style override
3. Check dark/light mode (mungkin toggle?
4. Screenshot dan compare dengan reference

### Issue: Icon tidak muncul?
**Solusi**:
1. Check CDN loading (F12 → Network)
2. Verify class name `ti ti-xxx` benar
3. Wait untuk Tabler Icons library load
4. Check internet connection
